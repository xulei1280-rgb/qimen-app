# Bazi UI Module

## Goal

Improve desktop/mobile layout and interaction without changing chart calculations.

## Primary File

Start with `bazi.html`. Locate only the component named in the request:

```powershell
rg -n "@media|renderBazi|renderPillarTable|renderCoreCards|renderBasicInfo|renderStructure|renderProfessionalLuckPanel|renderHistory" bazi.html check-bazi-page.js
```

## Allowed Scope

- HTML generated by the named render function.
- CSS and responsive media rules.
- Labels, buttons, tabs, folding, and layout-only state.
- Accessibility labels and mobile overflow fixes.

## Out of Scope

- Pillars, hidden stems, ten gods, strength, pattern, useful elements, luck dates,
  solar terms, or true-solar-time calculations.
- AI prompt semantics and saved AI context.

## Required Checks

```powershell
node .\check-bazi-page.js
```

After a visible change, verify the affected desktop and mobile view in the browser.
