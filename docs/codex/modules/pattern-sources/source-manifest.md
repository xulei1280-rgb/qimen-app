## 完整性规则

- 扫描页数、卷数或转录章节缺失时，清单必须明确写“缺卷”或“节录”，不得标为全文。
- 中国哲学书电子化计划中标注“版本暂缺”的文本只作全文检索提示，正式规则引用回到本目录扫描或有修订号的文库文本。
- 规则文档引用格式：`书名 / 卷或篇 / 本地文件 / 来源修订号`；没有篇章定位的摘句不得升级为默认规则。
