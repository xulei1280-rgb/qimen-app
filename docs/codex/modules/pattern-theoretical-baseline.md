## 用途与边界

本基准只比较同一套子平规则下的命盘结构相对位置，用于生成稳定的
`rawScore`、`percentile` 和 `baselineVersion`。它不使用名人、朋友、
产品用户或任何真实人物的人生轨迹，也不把现实成就作为训练标签。

理论百分位不证明命理对现实的解释力，也不能绕过成格、破格、救应、
作用链和清浊等子平硬门槛。正态分布和二八比例只能在此基准之上作
分档，不得反向修改单盘规则结论。

## 固定采样口径

- 时间范围：1984-02-04 至 2044-02-04，不含结束日，覆盖一个完整干支周期。
- 日期步长：每 11 天取样一次。11 与 60 互质，避免只抽到固定日柱余数。
- 时辰：每日取 `0, 2, 4, ..., 22` 十二个时辰中心点，避开时辰边界。
- 时区：`Asia/Shanghai`。
- 参考经度：东经 120°。
- 真太阳时：首期关闭；统一按东八区标准时生成，避免地点与均时差混入结构基准。
- 重复策略：按完整四柱去重；当前一个周期内无重复样本。
- 性别：原局格局层次不使用性别；性别只影响顺逆排运，因此不进入原局理论基准。

## 原始结构分

原始分范围为 0 至 100，`v4` 改用五维线性加权：

| 维度 | 权重 | 当前取值原则 |

| 成格基础 | 35% | 已成 85、成而有瑕 65、待成 45、已破 25；仅常格主线使用对应成格条件完成率，组合格和特殊格不套常格条件 |
| 作用链 | 20% | 无独立链 50；有链时按 `30 + 60 × 可达比例`，全通为 90 |
| 清浊去留 | 15% | 清至浊使用 90 至 10；已由病药家族认领的负面清浊按中性 50，避免重复扣分 |
| 病药净结算 | 20% | 无主要病点与全部化解同为 80；部分残病 55；存在未解病点 25 |
| 气势校正 | 10% | 中性 60；无病象且严格清秀 75；轻度、明显、严重病象为 45、30、10 |

格名不直接增加原始分。严格组合和特殊格只通过独立的质量下限、顶级资格和
同分顺序进入硬门槛。气势病象使用校正前的结构清浊，清秀气象不能抵消已成立的
气势病象。每个维度、权重、分值、加权贡献、证据和明细保存在
`patternRawScoreBreakdown`，所有贡献可以复算最终原始分。

该分数是项目工程排序，不是古籍原文数值，也不能单独作为“好命、坏命”的标签。

## 五档裁决

理论百分位先生成候选档位，再由子平成格硬门槛裁决最终档位：

| 百分位 | 候选档位 | 目标占比 |
| --- | --- | ---: |
| P0-P5 | 偏低 | 5% |
| P5-P20 | 中等 | 15% |
| P20-P80 | 偏高 | 60% |
| P80-P95 | 高 | 15% |
| P95-P100 | 顶级 | 5% |

硬门槛遵循以下顺序：

1. 未明确成格原则上封顶偏高；明确破格且无救封顶中等。
2. 常规格局已成取得中等下限，严格成格组合取得偏高下限。
3. 顶级必须同时满足候选百分位、严格成格、成格完整、作用可达、清纯且无残病和气势病象；格名本身不能直接顶档。
4. 病药按净结果结算：已解病点不重复扣档，部分化解只按残病限制，未解病点按实际严重度限制上限。
5. 同一原始分统一使用中位秩百分位，不随机拆分；子平硬门槛不同可以产生不同最终档位。

完整理论网格复算后的最终分布为：

| 最终档位 | 样本数 | 实际占比 |
| --- | ---: | ---: |
| 偏低 | 1,704 | 7.12% |
| 中等 | 3,076 | 12.86% |
| 偏高 | 14,775 | 61.78% |
| 高 | 3,984 | 16.66% |
| 顶级 | 377 | 1.58% |

顶级低于5%的目标比例，是因为百分位只提供候选资格，未通过严格硬门槛的
候选盘会回落到高或以下档位。这一结果不反向凑数，也不代表现实人口比例。

## 版本规则

- 日期范围、步长、时辰、去重方式、原始分公式或子平规则任一变化，都必须建立新基准版本。
- 旧版本不得静默覆盖；历史记录应保留原 `engineVersion`、`ruleVersion` 和 `baselineVersion`。
- `v2` 已完成 `PAT-101`、`PAT-102` 与 `PAT-113`：五档理论分布、子平硬门槛和同分政策均已冻结。
- `v3` 完成 `PAT-201` 首期：争合、完整三会三合、六合、冲与低阶刑害破先后裁决反馈到根损、作用链与层次，因此重新冻结分布。
- `v3` 的精确评分公式、文件哈希与恢复提交保存在
  [`pattern-score-backup.md`](pattern-score-backup.md)，在用户确认 `v4` 前不覆盖。
- `v4` 是 `PAT-208` 首个候选：格名奖励退出原始分，改用五维权重和可复算账本；
  子平规则版本、五档边界与硬门槛保持不变。基准生成改为两阶段，先生成新原始分
  直方图，再用该直方图重算候选档和最终档，避免沿用旧基准百分位。
