## 用途与边界

本基准只比较同一套子平规则下的命盘结构相对位置，用于生成稳定的
`rawScore`、`percentile` 和 `baselineVersion`。它不使用名人、朋友、
产品用户或任何真实人物的人生轨迹，也不把现实成就作为训练标签。

理论百分位不证明命理对现实的解释力，也不能绕过成格、破格、救应、
作用链和清浊等子平硬门槛。正态分布和二八比例只能在此基准之上作
分档，不得反向修改单盘规则结论。

## 固定采样口径

- 时间范围：1984-02-04 至 2044-02-04，不含结束日，覆盖一个完整干支周期。
- 日期步长：每 11 天取样一次。11 与 60 互质，避免只抽到固定日柱余数。
- 时辰：每日取 `0, 2, 4, ..., 22` 十二个时辰中心点，避开时辰边界。
- 时区：`Asia/Shanghai`。
- 参考经度：东经 120°。
- 真太阳时：首期关闭；统一按东八区标准时生成，避免地点与均时差混入结构基准。
- 重复策略：按完整四柱去重；当前一个周期内无重复样本。
- 性别：原局格局层次不使用性别；性别只影响顺逆排运，因此不进入原局理论基准。

原始分范围为 0 至 100，`v8` 继续使用六维线性加权。有效合会变气、根损和根力先反馈到
旺衰、十神力量与承载入口，再分别结算理论潜力、当前完成度和病药净结果：

| 维度 | 权重 | 当前取值原则 |
| --- | ---: | --- |
| 古籍格局潜力 | 20% | 只认仲裁后的一个主格；通过稳定规则验真的 S/A 级组合格与特殊格取 100/85；已完成月令立格的常格取 B 级 70，当前成败另由完成度与病药结算，未立格或待验取 0 |
| 成格完成度 | 25% | 已成 85、成而有瑕 65、待成 45、已破 25，并结合当前主格自己的成立检查完成率；已破格仍保留已经满足的正向条件，特殊格使用自身验真清单 |
| 作用链 | 20% | 无独立链 50；有链时按 `30 + 60 × 可达比例`，全通为 90 |
| 清浊去留 | 15% | 清至浊使用 90 至 10；已由病药家族认领的负面清浊按中性 50，避免重复扣分 |
| 病药净结算 | 15% | 无主要病点与全部化解同为 80；按未解病点和部分残病的残余强度比例递减；药神有根未透按部分救应，不再误作原局全无 |
| 气势校正 | 5% | 中性 60；无病象且严格清秀 75；轻度、明显、严重病象为 45、30、10 |

格名文字不直接增加原始分。只有经稳定规则 ID 验真的主格，才能取得一份带来源的
古籍格局潜力；兼格、线索和只修改显示文字均不叠加。财坏印等直接破点由当前完成度、
病药和硬门槛结算，不再把已经成立的月令格理论上限清零。流通、清浊、病药与气势分别结算，贵格不再通过额外的
“偏高下限”重复抬档。气势病象使用校正前的结构清浊，清秀气象不能抵消已成立的
气势病象。每个维度、权重、分值、加权贡献、证据和明细保存在
`patternRawScoreBreakdown`，所有贡献可以复算最终原始分。

该分数是项目工程排序，不是古籍原文数值，也不能单独作为“好命、坏命”的标签。
完整理论网格中的实际分数范围为 `40-89`，均值 `62.553144`，标准差 `10.115352`。

## 五档裁决

理论百分位先生成候选档位，再由子平成格硬门槛裁决最终档位：

| 百分位 | 候选档位 | 目标占比 |
| --- | --- | ---: |
| P0-P5 | 偏低 | 5% |
| P5-P20 | 中等 | 15% |
| P20-P80 | 偏高 | 60% |
| P80-P95 | 高 | 15% |
| P95-P100 | 顶级 | 5% |

硬门槛遵循以下顺序：

1. 未明确成格原则上封顶偏高；明确破格且无救封顶中等。
2. 常规格局已成、严格成格组合与严格特殊格均只取得中等下限；贵格潜力已经进入原始排序，不再设置第二个贵格抬档。
3. 顶级必须同时满足候选百分位、严格成格、成格完整、作用可达、清纯且无残病和气势病象；格名本身不能直接顶档。
4. 病药按净结果结算：已解病点不重复扣档，部分化解只按残病限制，未解病点按实际严重度限制上限。
5. 同一原始分统一使用中位秩百分位，不随机拆分；子平硬门槛不同可以产生不同最终档位。

完整理论网格复算后的最终分布为：

| 最终档位 | 样本数 | 实际占比 |
| --- | ---: | ---: |
| 偏低 | 1,269 | 5.31% |
| 中等 | 4,534 | 18.96% |
| 偏高 | 13,307 | 55.64% |
| 高 | 3,916 | 16.37% |
| 顶级 | 890 | 3.72% |

离散同分和严格成格闸门使顶级低于 5% 目标线；仍只有同时通过严格硬门槛的候选盘能够保留
顶级，未通过者会回落到高或以下档位。这一结果不随机拆分同分，也不代表现实人口比例。

## 版本规则

- 日期范围、步长、时辰、去重方式、原始分公式或子平规则任一变化，都必须建立新基准版本。
- 旧版本不得静默覆盖；历史记录应保留原 `engineVersion`、`ruleVersion` 和 `baselineVersion`。
- `v2` 已完成 `PAT-101`、`PAT-102` 与 `PAT-113`：五档理论分布、子平硬门槛和同分政策均已冻结。
- `v3` 完成 `PAT-201` 首期：争合、完整三会三合、六合、冲与低阶刑害破先后裁决反馈到根损、作用链与层次，因此重新冻结分布。
- 当前活动回滚点的精确公式、文件哈希与恢复提交保存在
  [`pattern-score-backup.md`](pattern-score-backup.md)；`v8` 是评分语义纠偏后的当前候选，不覆盖该备份。
- `v4` 是 `PAT-208` 首个候选：格名奖励退出原始分，改用五维权重和可复算账本；
  子平规则版本、五档边界与硬门槛保持不变。基准生成改为两阶段，先生成新原始分
  直方图，再用该直方图重算候选档和最终档，避免沿用旧基准百分位。
- `v5` 将可溯源的古籍格局潜力拆成独立维度，形成六维评分；严格组合与特殊格
  不再额外取得“偏高”质量下限。伤官配印、食神生财等必须通过现有成格规则，
  财坏印等反例退回常格潜力。特殊格改用自身的成立清单、顺势链和清浊语境，
  被其吸收的普通格病象只进入 `patternDisease.ignored`，不重复扣分。
- `v6` 修复真人反馈污染导致的食神制杀宽松成格，不改评分权重；所有受影响盘按新规则
  重新生成直方图、百分位和最终档位。
- `v7` 完成 `PAT-203` 原局首期，把有效合会变气与根损反馈到旺衰、十神根力、承载和特殊格验真；评分权重与理论分布不变，但单盘证据链和成立边界升级。
- `v8` 分离理论潜力与当前成败，保留破格中的已满足条件，并统一藏根药神与残病强度结算；
  评分分布必须重新审计，不得沿用 `v7` 的极值和离散度结论。
